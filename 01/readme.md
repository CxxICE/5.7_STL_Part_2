# Задача 1. Уникальный вектор

### Описание
Нужно реализовать функцию, которая удаляет все дубликаты, а не только подряд идущие, из std::vector.

### Пример правильной работы программы

```
[IN]: 1 1 2 5 6 1 2 4
[OUT]: 1 2 4 5 6
```
